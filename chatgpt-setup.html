ChatGPT API is NOT free. You need to add billing information to your OpenAI account before you can use the API. However, costs are very low for typical usage.

Typical costs:
  • ~$0.01 per scan (10 queries)
  • ~$1.00 for 100 scans
  • ~$10.00 for 1,000 scans

1. Create OpenAI Account

Go to platform.openai.com and create an account if you don't have one.

Create Account

2. Add Billing Information

In your OpenAI dashboard, go to Settings ‚Üí Billing and add a payment method. You can set usage limits to control costs.

Recommended: Set a monthly limit of $20 to start.

3. Generate API Key

Go to API Keys section and create a new secret key:

  1. Click "Create new secret key"
  2. Name it "Credli-ChatGPT" or similar
  3. Copy the key immediately (you won't see it again)
  4. It should start with "sk-proj-" or "sk-"
Get API Key

4. Test Your API Key

You can test if your API key works by running this command in terminal:

curl https://api.openai.com/v1/chat/completions \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_API_KEY_HERE" \ -d '{ "model": "gpt-4o-mini", "messages": [{"role": "user", "content": "Hello!"}], "max_tokens": 10 }'

Replace YOUR_API_KEY_HERE with your actual API key.

5. Ready to Use!

Once you have your API key, you can:

  • Add it to your environment variables as OPENAI_API_KEY
  • Test it with the Credli.ai ChatGPT integration
  • Monitor usage and costs in your OpenAI dashboard
